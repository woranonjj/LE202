## 3 ตัวอย่างรูปแบบพื้นฐานต่างๆในการสร้างรีโป้
### 3.1 # ตัวอักษรจะเล็กลงเรื่อยๆเมื่อเพิ่ม (#)
### ตัวอย่าง
# The largest heading
## The second largest heading
###### The smallest heading
### โค้ดที่ใช้
```
# The largest heading
## The second largest heading
###### The smallest heading
```
### 3.2Quoting text
### ตัวอย่าง
> Text that is a quote
### โค้ดที่ใช้
```
> Text that is a quote
```
